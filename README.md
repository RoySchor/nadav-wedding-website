# Hannah & Nadav — Wedding Website

[![website](https://img.shields.io/website?label=website&url=https%3A%2F%2Fschorwedding2027.com%2F)](https://schorwedding2027.com/)

The official wedding website for Hannah & Nadav's wedding weekend at Camp Lindenmere, June 11–13, 2027.

## Pages

- **Home** — Hero photo and welcome message
- **Weekend Agenda** — Full itinerary for Friday, Saturday (wedding day), and Sunday
- **Travel & Lodging** — Directions, transit options, and cabin info
- **FAQs** — Meals, packing tips, children policy
- **RSVP** — Coming soon

## Tech Stack

- React 19 + Vite
- React Router (client-side routing)
- Google Fonts (Cormorant Garamond + Montserrat)
- Deployed to GitHub Pages via GitHub Actions

## Development

```bash
npm install
npm run dev
```

## Deployment

Pushes to `main` automatically deploy to GitHub Pages via the workflow in `.github/workflows/deploy.yml`.
